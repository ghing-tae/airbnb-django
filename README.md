# airbnb-django-practice
> Cloning Airbnb with Python, Django, Tailwind and more... 🐍
## pipenv shell
```
pipenv shell은 나를 버블 안으로 데려다 주는 일을 한다. 
```
> django작업 환경에서 새롭게 작업을 시작하면 pipenv shell 필수
---

## Django
```
Django에서 새로운 프로젝트를 만들려면 작업 파일 에서
"django-admin startproject config"
한다.
```
> 생성되어지는 config 폴더의 이름을 변경하고 안에있는 config 폴더와 manage.py를 밖으로 꺼낼 것 이기 때문이다.

> python 버전은 3.x 로 사용하자 pipenv안에 작업 환경을 만들 것 이기 떄문에 pipenv 인지 확인하자.

```
pipenv안에서 서버를 실행하기 위해선
python manage.py runserver 
```
